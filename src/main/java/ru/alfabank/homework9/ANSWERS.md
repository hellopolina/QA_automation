1. Нужно ли было впринципе писать свой собственный генератор с фамилиями? Было бы проще использовать генератор из datafaker?
Не стала писать генератор, сделала через RandomStringUtils, у которого метод randomAlphabetic возвращает просто набор букв.
Конечно, через datafaker это было бы сделать лучше, т.к. результат - человекочитаемая фамилия.
2. Какая из двух библиотек из этого задания может больше пригодится при написании автотестов - jfiglet или datafaker?
Для автотестов больше пригодится datafaker. Например, для генерации перс. данных по клиентам.
3. Каким образом JAR-файлы этих библиотек попали к нам в проект?
При указании библиотек в build.gradle gradle обращается к удалённому репозиторию и скачивает требуемые JAR-файлы.
